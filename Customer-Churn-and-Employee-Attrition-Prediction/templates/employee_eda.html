KDEplots

KDE (Kernel Density Estimation) plot is a type of visualization technique used to represent the probability density function of a continuous random variable.

A KDE plot displays the density estimate of a variable, which is a smoothed version of the histogram. In a KDE plot, a kernel function is placed at each data point and then smoothed out to create a continuous density estimate.

A bivariate KDE plot is a type of visualization that shows the joint distribution of two variables using a kernel density estimate.

In a bivariate KDE plot, a two-dimensional density estimate is created by placing a kernel function at each point in the two-dimensional space. These kernel functions are then smoothed out to create a continuous surface that represents the density of the data.

The plot shows the joint distribution of the two variables, with contours or shading indicating the areas of highest density. This can help identify patterns and relationships between the two variables, such as correlation or clustering.

Pairplot Visualisation

Seaborn's pairplot is a type of visualization that allows for the creation of a grid of plots, where each variable in a dataset is plotted against every other variable. This creates a visual representation of the pairwise relationships between the variables, making it easier to identify patterns and correlations in the data.

Each plot in the pairplot displays a scatterplot of the two variables and a histogram of each variable's distribution. Additionally, diagonal plots show a kernel density estimate of the distribution of each variable. Pairplots can also be customized with various options, including adding regression lines, changing the color scheme, and displaying categorical variables using different symbols or colors. Overall, pairplot is a powerful tool for exploratory data analysis, as it allows for quick and comprehensive visual inspection of relationships between variables in a dataset.
